About

I’ve been wearing a Garmin tracker for over two years now and finally decided to something with the data. I’ve ported over all of my historic activity data to Strava and using the relatively new rStrava package ([Github])(https://github.com/fawda123/rStrava){target=“_blank“} I can easily download all of the data and play with it.

For this, I’m going to convert each year of data into an interactive heatmap that tracks my activity. This will be done with the excellent highcharter package from jkunst. The only issue with highcharter is that it doesn’t have faceting capaibilities.
