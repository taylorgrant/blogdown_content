Running data from Strava

I’ve been wearing a Garmin tracker for over two years now and finally decided to something with the data. I’ve ported over all of my historic activity data to Strava and using the relatively new rStrava package Github I can easily download all of the data. All the information for setting up the application for the Strava API is on the github page for rStrava.

For this, I’m going to convert each year of data into an interactive heatmap that tracks my activity. This will be done with the excellent highcharter package from jkunst. The only issue with highcharter is that it doesn’t have faceting capaibilities.
